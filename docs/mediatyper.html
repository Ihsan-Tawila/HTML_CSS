Media typer

Intro

medietyper er forskjellige css-stilark for en nettside for flere typer enheter. når man skal designe nettside kan man bruke flere css-stilark hvor man kan velge hvilken stilark skal bruke.

hvordan man bruker Media typer

Det er for tiden to måter å spesifisere medieavhengighet for stilark:

Spesifiser målmediet fra et stilark med @media eller @import at-rules.

 @import url("fancyfonts.css") screen;
	eller
 @media print {
    body { font-size: 10pt }
    }	
 @media screen {
    body { font-size: 12pt }
    }
 @media screen, print {
    body { line-height: 1.2 }
	}
 }

Spesifiser målmediet i dokumentspråket. For eksempel, i HTML 5, spesifiserer "media" -attributtet på LINK-elementet målmediet til et eksternt stilark:

 <!DOCTYPE HTML>
 <HTML>
   <HEAD>
      <TITLE>Link to a target medium</TITLE>
      <LINK REL="stylesheet" TYPE="text/css" 
	 MEDIA="print, handheld" HREF="foo.css">
   </HEAD>
   <BODY>
      <P>The body...
   </BODY>
 </HTML>

De forskjellige mediatypene

Det er mange forskjellig mediatyper kan markeres:

  • All: brukes til alle enheter
  • Braille: brukes for punk taktil tilbakemelding innretninger
  • Embossed: brukes til sideskrift
  • Handheld: brukes for håndholdte enheter
  • print: Brukes til skrivere
  • projection: brukes for prosjektert presentasjon
  • screen: brukes for fargeskjerm
  • speech: brukes for tale synthesizere
  • tty: brukes for media ved hjelp av et rutenett med fast tonehøyde
  • TV: brukes for TV

Fordeler og ulemper med å bruke medietyper

Noen medier har store fordeler, for eksempel en skriver "printer" media typen hjelper ved å skille på hvilket stilark skal brukes når man ser på skjarmen og når man skal printer utm nettsiden. Gjennom dette kan vi slette uviktige deler på siden, for eksempel annonser. Vi kan også redusere fargene som brukes på websiden for å spare blekk og andre fordeler.

noen av mediatyper blir nesten dødt f.eks medie typen handhold og det er på grunn av responsivt design som gjør at nettsiden tilpasse seg når nettleser viduet blir smallere eller større og derfor trenges ikke et spesialt stilark for det forskjellige enhet stølelser. Tv medie typen også er nesten dødt pågrunn av tv-ene nå tildags er ganske høy oppløsning som gjør ar det er mindre behov til tilpasse dette formatet.
